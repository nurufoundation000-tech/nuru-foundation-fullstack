What is JSON?

JSON (JavaScript Object Notation) is a lightweight format for storing and transporting data. It's the standard format for API data.

JSON Structure

{ "name": "Alex", "age": 25, "isStudent": true, "courses": ["HTML", "CSS", "JavaScript"], "address": { "city": "Nairobi", "country": "Kenya" } }
Did You Know?

JSON is language-independent - it works with Python, Java, PHP, and many other languages!

Working with APIs

APIs (Application Programming Interfaces) let your code communicate with external services to get or send data.

Fetch API

// Basic fetch request fetch("https://api.example.com/data") .then(response => response.json()) .then(data => { console.log(data); }) .catch(error => { console.log("Error:", error); });

Async/Await (Modern Syntax)

async function getData() { try { const response = await fetch("https://api.example.com/data"); const data = await response.json(); console.log(data); } catch (error) { console.log("Error:", error); } } getData();

Displaying Dynamic Data

Let's build a dynamic user card that displays data from an API.

HTML Container

<div id="userContainer"> <div class="loading">Loading...</div> </div>

JavaScript to Fetch and Display

async function loadUsers() { const container = document.getElementById("userContainer"); try { const response = await fetch( "https://jsonplaceholder.typicode.com/users" ); const users = await response.json(); // Clear loading text container.innerHTML = ""; // Create cards for each user users.forEach(user => { const card = document.createElement("div"); card.className = "user-card"; card.innerHTML = ` <h3>${user.name}</h3> <p>${user.email}</p> <p>${user.address.city}</p> `; container.appendChild(card); }); } catch (error) { container.innerHTML = "<p>Failed to load users</p>"; } } loadUsers();
Important

Always handle errors with try/catch when making API calls. Network requests can fail for many reasons!

Building a Dynamic Page

Let's create a simple news feed that fetches and displays articles.

Step-by-Step

  1. Create an HTML container for articles
  2. Write JavaScript to fetch from a news API
  3. Map through the data and create HTML elements
  4. Handle loading and error states
// Fetch and display articles const articles = [ { "title": "JavaScript is Awesome", "summary": "Learn JS today" }, { "title": "Web Dev Tips", "summary": "Best practices" } ]; articles.forEach(article => { const element = document.createElement("article"); element.innerHTML = ` <h2>${article.title}</h2> <p>${article.summary}</p> `; document.getElementById("feed").appendChild(element); });

Best Practices

  • Always handle loading states while fetching
  • Show user-friendly error messages
  • Validate data before using it
  • Don't expose API keys in client-side code
  • Use async/await for cleaner asynchronous code
Pro Tip

Use JSONPlaceholder (jsonplaceholder.typicode.com) for practice - it's a free fake API for testing!
