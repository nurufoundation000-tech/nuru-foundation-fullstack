Events & Interactivity

Learning Objectives

  • Understand JavaScript event handling
  • Add click handlers and other event listeners
  • Work with forms and user input
  • Create interactive web experiences

What are Events?

Events are actions that happen in the browser - clicks, key presses, page loads, form submissions, and more. JavaScript can respond to these events to create interactive pages.

Common Events

  • click: When user clicks an element
  • mouseover/mouseout: When mouse enters/leaves
  • keydown/keyup: When keys are pressed
  • submit: When a form is submitted
  • load: When page finishes loading

Event Listeners

Event listeners watch for events and run code when they occur.

addEventListener Method

const button = document.querySelector("#myButton"); button.addEventListener("click", function() { alert("Button clicked!"); });

Arrow Function Version

button.addEventListener("click", () => { button.textContent = "Clicked!"; });

Working with Forms

Forms are key to user interaction. Let's create a simple form handler.

HTML Form

<form id="contactForm"> <input type="text" id="name" placeholder="Your name"> <input type="email" id="email" placeholder="Your email"> <button type="submit">Submit</button> </form>

JavaScript Form Handler

const form = document.getElementById("contactForm"); form.addEventListener("submit", (e) => { e.preventDefault(); // Stop page reload const name = document.getElementById("name").value; const email = document.getElementById("email").value; console.log(`Name: ${name}, Email: ${email}`); alert("Form submitted!"); });
Important

Always use e.preventDefault() in form handlers to prevent the page from refreshing, which is the default browser behavior.

Practical Examples

Toggle Content

const toggleBtn = document.getElementById("toggleBtn"); const content = document.getElementById("content"); toggleBtn.addEventListener("click", () => { if (content.style.display === "none") { content.style.display = "block"; } else { content.style.display = "none"; } });

Counter Example

let count = 0; const display = document.getElementById("countDisplay"); document.getElementById("increment").addEventListener("click", () => { count++; display.textContent = count; });

Event Object

Event handlers receive an event object with useful properties and methods.

element.addEventListener("click", (event) => { console.log(event.type); // "click" console.log(event.target); // The clicked element console.log(event.clientX); // Mouse X position console.log(event.clientY); // Mouse Y position });
Pro Tip

Use event.target to get the exact element that triggered the event - very useful when delegating events to parent elements.
