🛠️ 2. Lab 1 — System Updates (Deep Dive)

Updating isn't just about clicking a button. It's about ensuring nothing is left behind.

[IMAGE: Screenshot showing Windows Update Settings with "Check for Updates" highlighted]

The Pro Routine:

  1. Open Settings > Windows Update.
  2. Click Check for updates.
  3. Crucial: Click "Advanced Options" > "Optional Updates." Check for Driver updates here. (Outdated drivers are a common hacker entry point!)
  4. Install all and Restart.

🔍 3. Lab 2 — Checking Patch Level

A professional technician always verifies the work. You need to know exactly how "modern" your defense is.

Execution:

  • Type winver in the Start Menu and press Enter.
  • Record your Version: (e.g., Version 22H2 or 23H2).
  • Check Update History: Look for any "Failed to install" messages. If an update fails, your system is still vulnerable!
Why? If a hacker knows you are on Version 21H1, they can use an exploit from 2022 that hasn't been patched on your machine yet.

🛡️ 4. Lab 3 — Full Hardening Checklist (The Clean-up)

Now we reduce the Attack Surface. If you don't use it, remove it!

Remove Bloatware: Go to "Add or Remove Programs." Uninstall games, trial software, and apps you don't use. Each one is a potential door for a hacker.
Disable Startup Apps: Open Task Manager (Ctrl+Shift+Esc). Disable apps that don't need to start automatically.
[IMAGE: Task Manager showing the "Startup" tab and disabling unnecessary apps]

🔐 4. Lab 3 — Full Hardening Checklist (The Locks)

Configure the "Active" defenses of your machine.

User Account Control (UAC): Set to "Always Notify." This stops apps from installing themselves without your permission.
Screen Lock: Set your screen to lock after 2–5 minutes of inactivity. Physical security is cybersecurity!
Privacy Settings: Disable "Advertising ID" and "Location" for apps that don't need it.
Auto-Updates: Ensure "Pause updates" is NOT active.
