🖥️ 2. What is a Virtual Machine (VM)?

Imagine you want to practice painting a wall, but you don't want to ruin your house. You buy a toy house to practice on first. A VM is that toy house!

Benefits of VMs:

  • 🛡️ Safe Testing: If you break the VM, your real computer is fine.
  • ☣️ Malware Lab: You can open "dangerous" files inside a VM safely.
  • 🐧 Try Anything: You can run Linux, Windows, or old systems all at once.

🧪 Step-by-Step Installation

Follow these steps exactly to build your Linux machine:

1 Open VirtualBox and click the "New" (blue star) button.

2 Name it: Type "My Linux Lab." Choose your ISO file in the dropdown.

3 The Brain (RAM): Give it 2GB or 4GB (Slide the bar to the green zone).

4 The Closet (Storage): Give it 25GB or 40GB of "Virtual Disk Space."

5 Start: Click the Green Arrow! Follow the on-screen prompts (Choose Language > Install).

⌨️ Lab 2 — Speaking "Terminal"

Open the app called Terminal in Linux. This is where you type commands to the Boss.

🔹 Navigation Commands:

  • pwd → "Where am I?" (Print Working Directory)
  • ls → "Show me the stuff!" (List files in the folder)
  • cd → "Move me!" (Change Directory to another folder)
  • clear → "Clean the desk!" (Wipes the screen clean)

📂 Managing Files

In the terminal, you can create and destroy files instantly!

  • mkdir MyFolder → Creates a new Folder.
  • touch secret.txt → Creates a new Empty File.
  • rm secret.txt → Deletes a file (Careful! No recycle bin here).
  • cp file.txt backup.txt → Copies a file.
  • mv file.txt Documents/ → Moves a file to a folder.

🔍 System & Info Commands

Use these to see how "healthy" your Linux machine is:

  • whoami → Shows which user you are logged in as.
  • uname -a → Tells you the version of Linux you are running.
  • df -h → Shows how much Disk Space is left.
  • free -h → Shows how much Memory (RAM) is being used.
  • history → Shows every command you have typed so far!
