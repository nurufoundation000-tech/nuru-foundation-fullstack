1. Accessing the Control Panel

To secure Maca's network, we first need to find the "Default Gateway"—the router's home address.

How to find it:
Windows: Type ipconfig in Command Prompt.
Mac/Linux: Type ifconfig in Terminal.

Common Gateways: 192.168.0.1, 192.168.1.1, 10.0.0.1

Step-by-Step Access

Maca opens his browser and enters the Gateway IP. Now, he faces the login screen.

  • 1. Open Browser: Enter the IP in the URL bar.
  • 2. Login Prompt: Routers ship with "Public Knowledge" passwords.
Dangerous Default Credentials:
Username: admin / Password: admin
Username: admin / Password: password

If these aren't changed, Maca's network is open to anyone!

🛡️ Config 1 & 2: The Foundation

  • 1. Change Admin Credentials: Replace the factory password with something unique.
    Example: Router!Fortress@2024#Secure
  • 2. Update Firmware: Think of this as the router's "Vaccine." It patches security holes.
    Pro Tip: Always backup settings before updating firmware!

    • 🛡️ Config 3: Wi-Fi Shielding

    Maca’s kids need Wi-Fi, but it must be encrypted. Here is how we choose:

    Setting Secure Choice Why?
    SSID Name HiddenVortex (No names!) Prevents targeted attacks.
    Encryption WPA3 or WPA2 (AES) WEP is crackable in minutes.
    Password 12+ Characters Resists brute-force guessing.
    WPS DISABLE WPS has massive security flaws.

    🛡️ Config 4: The Firewall Guard

    The firewall acts as Maca's digital security guard, checking every "visitor" (data packet) at the door.

    • ✅ SPI Firewall: Only allows traffic you specifically asked for.
    • ✅ DoS Protection: Prevents attackers from "flooding" and crashing Maca's internet.
    • ✅ Block WAN Ping: Makes the router "invisible" to hackers scanning the web.

    🛡️ Config 5: Guest Isolation

    When Maca has visitors, he shouldn't give them his main password. Why?

    The "Guest House" Strategy:

    Creating a Guest Network isolates visitors. If a visitor's phone has malware, it cannot "jump" to Maca's private family computers.

    Check: Ensure "AP Isolation" is enabled so guests can't see each other either.
