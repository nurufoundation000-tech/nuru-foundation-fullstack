1. Seeing the Wireless World

Your computer sees much more than just a list of names. It detects security protocols and precise signal strengths.

Windows Users (Command Prompt):
netsh wlan show networks mode=bssid
Mac Users:
Hold Option key + Click the Wi-Fi icon in the menu bar.

What to note: Authentication type (WPA2 vs WEP) and Signal Strength (e.g., 85%).

2. Advanced Analysis Tools

To go deeper, Maca’s uses professional-grade software to "sniff" packets and map signal coverage.

  • Wireshark (Packet Sniffing): Visualizes raw data traveling through the air. Use filter wlan to isolate Wi-Fi traffic.
  • WiFi Analyzer / NetSpot: Creates a "Heat Map" of signal coverage.
Your Task: Install an analyzer, scan your environment, and identify the weakest encryption type in range.

3. Identifying Insecure Networks

Not all Wi-Fi is created equal. Use this guide to assess Maca’s's neighbors:

Security Type Status Time to Crack Real-World Example
WEP ☠️ DEAD 1-5 Minutes Legacy devices / old hardware
WPA ⚠️ VULNERABLE Hours to Days Small, unpatched coffee shops
WPA2 ✅ SECURE Years (Brute Force) Modern home networks
WPA3 🛡️ HIGHLY SECURE Currently Unbreakable Enterprise/Government/New Tech

4. Wireless Attack Scenarios

How do attackers trick Maca’s's family? They exploit the "trust" of wireless connections.

  • 😈 Evil Twin: A fake network with the same name as a real one (e.g., "Airport_Free").
  • 🚧 Deauthentication: Forcing Maca’s's device off the real network so it connects to the hacker's "Evil Twin."
  • 🔑 WPS PIN Attack: Guessing a small numeric PIN to bypass WPA2 passwords.

DEFENSE: Use a VPN and Disable WPS immediately!
