CSS Fundamentals

Learning Objectives

  • Understand what CSS is and how it styles HTML elements
  • Learn different types of CSS selectors
  • Master common CSS properties for styling
  • Learn the basics of Flexbox for layout

What is CSS?

CSS (Cascading Style Sheets) is a language used to describe how HTML elements should be displayed. While HTML provides structure, CSS handles the visual presentation.

Why Use CSS?

  • Separation of Concerns: Keep styling separate from content
  • Consistency: Apply styles across multiple pages
  • Flexibility: Create responsive designs
  • Efficiency: One style sheet controls many pages

CSS Selectors

Selectors target HTML elements to apply styles. Here are the main types:

Element Selector

p { color: blue; font-size: 16px; }

Class Selector

.highlight { background-color: yellow; }

ID Selector

#header { background: navy; }

Universal Selector

* { margin: 0; padding: 0; }

Common CSS Properties

Colors & Backgrounds

.box { color: #333; background-color: white; background-image: url('bg.jpg'); }

Typography

.text { font-family: Arial, sans-serif; font-size: 18px; font-weight: bold; text-align: center; }

Box Model

.container { margin: 20px; padding: 15px; border: 1px solid black; width: 300px; height: 200px; }

Introduction to Flexbox

Flexbox is a powerful CSS layout module for arranging items in rows or columns.

Container Properties

.flex-container { display: flex; flex-direction: row; /* row, column */ justify-content: center; /* horizontal */ align-items: center; /* vertical */ gap: 20px; }

Item Properties

.flex-item { flex: 1; /* grow to fill space */ }
Pro Tip

Flexbox is perfect for centering elements. Use justify-content and align-items both set to "center" for perfect centering!

Connecting CSS to HTML

Inline Styles

<p style="color: red;">Red text</p>

Internal CSS

<style> p { color: blue; } </style>

External CSS (Recommended)

<link rel="stylesheet" href="style.css">
Important

Always prefer external CSS files. They can be shared across multiple pages and cached by browsers for better performance.
