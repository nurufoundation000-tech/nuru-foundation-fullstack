Introduction to HTML

Learning Objectives

  • Understand what HTML is and its role in web development
  • Learn the basic structure of an HTML document
  • Identify and use common HTML tags
  • Create semantic HTML elements

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML describes the structure of a webpage using a system of tags and attributes.

Why Learn HTML?

  • Foundation of Web: Every website you see uses HTML
  • Easy to Learn: Simple syntax that's great for beginners
  • Essential Skill: Required for all web developers
  • Universal: Works on all browsers and devices
Did You Know?

HTML was created by Tim Berners-Lee in 1991. It's now maintained by the World Wide Web Consortium (W3C).

Basic HTML Structure

Every HTML document follows a standard structure. Let's break it down:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Page</title> </head> <body> <!-- Your content goes here --> </body> </html>

Understanding Each Part

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element that contains all other elements
  • <head> - Contains metadata (title, charset, links)
  • <body> - Contains all visible content

Common HTML Tags

HTML uses tags to mark up content. Here are the most common ones:

Headings

<h1>Main Heading</h1> <h2>Subheading</h2> <h3>Section Title</h3>

Paragraphs

<p>This is a paragraph of text.</p>

Links

<a href="https://google.com">Click here</a>

Images

<img src="image.jpg" alt="Description">

Semantic HTML

Semantic HTML uses meaningful tags that describe their content. This improves accessibility and SEO.

  • <header> - For page headers
  • <nav> - For navigation menus
  • <main> - For main content
  • <article> - For standalone content
  • <footer> - For page footers
Best Practice

Always use semantic HTML elements instead of generic <div> tags when possible. It makes your code more meaningful and accessible.

Practice Exercise

Create an HTML page with:

  1. A main heading (h1)
  2. A paragraph describing yourself
  3. A link to your favorite website
  4. An image
Remember

HTML tags usually come in pairs - an opening tag and a closing tag. The closing tag has a forward slash (/) before the tag name.
