Project Structure

A well-organized React project makes development easier as your app grows.

Recommended Folder Structure

my-app/ ├── src/ │ ├── components/ // Reusable components │ │ ├── Header.jsx │ │ ├── Footer.jsx │ │ └── Card.jsx │ ├── pages/ // Page components │ │ ├── Home.jsx │ │ └── About.jsx │ ├── hooks/ // Custom hooks │ ├── styles/ // CSS files │ ├── App.jsx // Main App │ └── main.jsx // Entry point ├── public/ // Static files └── package.json

Building a Complete App

Let's build a product listing app combining all concepts.

App.jsx - Main Component

import { useState } from 'react'; import Header from './components/Header'; import ProductCard from './components/ProductCard'; import './App.css'; const initialProducts = [ { id: 1, name: "Laptop", price: 999, image: "💻" }, { id: 2, name: "Phone", price: 699, image: "📱" }, { id: 3, name: "Headphones", price: 199, image: "🎧" }, ]; export default function App() { const [products] = useState(initialProducts); return ( <div className="app"> <Header title="My Store" /> <main className="products"> {products.map(product => ( <ProductCard key={product.id} product={product} /> ))} </main> </div> ); }

ProductCard Component

export default function ProductCard({ product }) { return ( <div className="product-card"> <span className="image">{product.image}</span> <h3>{product.name}</h3> <p className="price">${product.price}</p> <button>Add to Cart</button> </div> ); }

Styling React Apps

There are many ways to style React applications. We'll use standard CSS.

App.css

.app { max-width: 1200px; margin: 0 auto; padding: 20px; } .products { display: grid; grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); gap: 20px; } .product-card { background: white; padding: 20px; border-radius: 12px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); text-align: center; } .product-card .image { font-size: 4rem; }
Pro Tip

CSS Modules or styled-components are popular alternatives that scope styles to components. Try them as you advance!

Deploying to Netlify

Netlify makes deploying React apps incredibly easy.

Steps to Deploy

  1. Push your code to GitHub
  2. Go to netlify.com and sign up
  3. Click "Add new site" → "Import an existing project"
  4. Select your GitHub repository
  5. Click "Deploy site"

Alternative: Drag and Drop

  1. Run npm run build in your project
  2. Go to the "dist" or "build" folder
  3. Drag the folder to Netlify's drop zone
Bonus: Vercel

Vercel is another great option for deploying React apps. Just connect your GitHub repo and it auto-deploys!
