Introduction to React

Learning Objectives

  • Understand what React is and why it's popular
  • Learn about components and JSX
  • Create your first React component
  • Understand the virtual DOM concept

What is React?

React is a JavaScript library for building user interfaces. Developed by Facebook, it's the most popular front-end framework used today.

Why Learn React?

  • Component-Based: Build reusable pieces of UI
  • Virtual DOM: Fast and efficient updates
  • Declarative: Describe what you want, not how
  • Large Ecosystem: Libraries, tools, and community
Did You Know?

React is used by Facebook, Instagram, Netflix, and many other major companies!

Setting Up React

The easiest way to create a React app is using Create React App or Vite.

Using Vite (Recommended)

# Create a new React project npm create vite@latest my-app -- --template react # Navigate to the folder cd my-app # Install dependencies npm install # Start the development server npm run dev
Pro Tip

Vite is much faster than Create React App for project setup and development!

Components

Components are the building blocks of React applications. They let you split the UI into independent, reusable pieces.

Function Component

function Welcome() { return ( <div> <h1>Hello, World!</h1> </div> ); }

Arrow Function Version

const Welcome = () => { return ( <h1>Hello, World!</h1> ); };

Using the Component

function App() { return ( <div> <Welcome /> <Welcome /> </div> ); }

JSX - JavaScript XML

JSX is a syntax extension that lets you write HTML-like code in JavaScript. It looks like HTML but is actually JavaScript.

JSX Rules

  • Return a single root element
  • Close all tags (including self-closing)
  • Use camelCase for attributes
  • Use className instead of class
// JSX looks like HTML but is JavaScript const element = <h1 className="title">Hello!</h1>; // You can embed JavaScript with curly braces const name = "Alex"; const greeting = <p>Hello, {name}</p>;
Important

JSX is not valid JavaScript - it gets transformed to regular JavaScript function calls by a transpiler (like Babel) before running in the browser.

Creating a Simple App

Let's create a simple React app with multiple components.

App.jsx

// Header Component function Header() { return <header><h1>My React App</h1></header>; } // Card Component function Card(props) { return ( <div className="card"> <h2>{props.title}</h2> <p>{props.description}</p> </div> ); } // Main App Component export default function App() { return ( <div> <Header /> <Card title="React" description="A UI library" /> <Card title="Components" description="Building blocks" /> </div> ); }
