State & Props

Learning Objectives

  • Understand React state and how to use it
  • Learn to pass data with props
  • Handle user interactions and updates
  • Understand the component lifecycle

What is State?

State is data that changes over time in your component. When state changes, React automatically re-renders the component.

useState Hook

import { useState } from 'react'; function Counter() { const [count, setCount] = useState(0); return ( <div> <p>Count: {count}</p> <button onClick={() => setCount(count + 1)}> Increment </button> </div> ); }
Hooks

useState is a "hook" - special functions that let you use state and other React features in function components.

What are Props?

Props (short for properties) are how you pass data from parent components to child components.

Passing Props

// Parent component function App() { return <Greeting name="Alex" age={25} />; } // Child component receiving props function Greeting(props) { return ( <div> <h1>Hello, {props.name}!</h1> <p>You are {props.age} years old.</p> </div> ); }

Using Destructuring

// Clean way to access props function Greeting({ name, age }) { return <p>Hello, {name}!</p>; }

Handling Events

React handles events with props like onClick, onChange, onSubmit.

function LoginForm() { const [username, setUsername] = useState(""); const handleSubmit = (e) => { e.preventDefault(); alert(`Logged in as ${username}`); }; return ( <form onSubmit={handleSubmit}> <input type="text" value={username} onChange={(e) => setUsername(e.target.value)} /> <button type="submit">Login</button> </form> ); }
Important

Always use controlled components - where the input value is controlled by React state, not the other way around.

Component Lifecycle

Function components don't have traditional lifecycle methods, but useEffect hook lets you run code at specific times.

useEffect Hook

import { useState, useEffect } from 'react'; function DataFetcher() { const [data, setData] = useState(null); useEffect(() => { // Runs once when component mounts fetch('/api/data') .then(res => res.json()) .then(result => setData(result)); }, []); // Empty array = run once return <p>{data ? data.message : "Loading..."}</p>; }
Pro Tip

useEffect with an empty dependency array [] runs only once on mount. With dependencies, it runs when those values change.
