Word Processing with MS Word

Learning Objectives

  • Create and save new documents in Microsoft Word
  • Apply text formatting (bold, italic, fonts, colors)
  • Insert and format tables in documents
  • Add images and shapes to enhance documents
  • Use page layout and printing options

Getting Started with MS Word

Microsoft Word is the world's most popular word processing software. It's used for creating letters, reports, resumes, and many other types of documents.

Creating a New Document

  • Open Microsoft Word from the Start menu or desktop shortcut
  • Click "Blank Document" to start a new page
  • Your document opens with a cursor ready for typing
  • Save your document using Ctrl+S or File > Save As
Did You Know?

The default file extension for Word documents is .docx, which replaced the older .doc format!

Text Formatting

Formatting makes your documents more readable and professional. Use the formatting toolbar or keyboard shortcuts.

Basic Formatting Options

  • Bold: Ctrl+B - Makes text thicker and darker
  • Italic: Ctrl+I - Slants text for emphasis
  • Underline: Ctrl+U - Draws a line under text
  • Font Size: Change text size from the dropdown (default is 11pt)
  • Font Family: Choose different text styles like Arial, Times New Roman
  • Text Color: Change the color of your text
Pro Tip

Use the Format Painter (paintbrush icon) to quickly copy formatting from one text to another!

Working with Tables

Tables are perfect for organizing information in rows and columns. They're great for schedules, data, and comparisons.

Creating and Formatting Tables

  • Insert Table: Click Insert > Table and drag to select rows and columns
  • Add Rows/Columns: Right-click and choose Insert Above/Below or Left/Right
  • Merge Cells: Select cells and click Merge Cells to combine them
  • Table Styles: Use pre-designed table styles for professional looks
  • Adjust Columns: Drag the gridlines to resize columns
Watch Out

Always press Tab to move between table cells - pressing Enter creates a new row instead!

Adding Images and Graphics

Visual elements make your documents more engaging. Word allows you to insert pictures, shapes, and SmartArt.

Inserting Visual Elements

  • Pictures: Insert > Pictures to add photos from your computer
  • Online Pictures: Search Bing for images directly in Word
  • Shapes: Insert > Shapes to add arrows, boxes, and other shapes
  • SmartArt: Create professional diagrams like process flows
  • Text Boxes: Add floating text anywhere on the page
Remember

Use the picture formatting tools to crop, resize, and add effects to images!

Practice Exercise

Create a Professional Letter

  1. Create a new blank document
  2. Add a header with your name and address (use right-align)
  3. Format the title in bold, 16pt font
  4. Insert a 3-column table with dates, topics, and notes
  5. Add a relevant image to enhance the document
  6. Save your document as "MyLetter.docx"
