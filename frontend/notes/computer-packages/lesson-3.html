Spreadsheets with MS Excel

Learning Objectives

  • Navigate the Excel interface and understand workbooks, worksheets, and cells
  • Enter and format data in spreadsheets
  • Use basic formulas and functions for calculations
  • Create and customize charts to visualize data
  • Apply basic data analysis techniques

Excel Basics

Microsoft Excel is a powerful spreadsheet program used for organizing, calculating, and analyzing data. It's essential for business, finance, and data management.

Understanding the Excel Interface

  • Workbook: A file containing multiple worksheets (like a book)
  • Worksheet: A single spreadsheet with rows and columns
  • Cell: The intersection of a row and column (e.g., A1, B5)
  • Column Letters: A, B, C... (vertical)
  • Row Numbers: 1, 2, 3... (horizontal)
  • Formula Bar: Shows and edits cell contents
Did You Know?

A single Excel worksheet can have over 1 million rows and 16,000 columns - that's a lot of data!

Working with Formulas

Formulas are what make Excel powerful. They perform calculations on your data automatically.

Basic Formulas

  • Always start with = to tell Excel you're entering a formula
  • Addition: =A1+B1 adds values in cells A1 and B1
  • Subtraction: =A1-B1 subtracts B1 from A1
  • Multiplication: =A1*B1 multiplies the values
  • Division: =A1/B1 divides A1 by B1

Common Functions

=SUM(A1:A10) - Adds all values from A1 to A10
=AVERAGE(A1:A10) - Calculates the average
=MAX(A1:A10) - Finds the largest value
=MIN(A1:A10) - Finds the smallest value
=COUNT(A1:A10) - Counts cells with numbers
Pro Tip

Double-click the small square in the bottom-right of a cell to copy formulas down quickly!

Creating Charts

Charts help you visualize data and make it easier to understand trends and patterns.

Types of Charts

  • Column Chart: Best for comparing categories
  • Line Chart: Shows trends over time
  • Pie Chart: Shows parts of a whole (percentages)
  • Bar Chart: Horizontal comparison of categories
  • Area Chart: Emphasizes the magnitude of change

Creating a Chart

  • Select the data you want to include in the chart
  • Click Insert > Charts and choose your chart type
  • Use Chart Tools to customize colors, titles, and labels
  • Move or resize the chart as needed
Watch Out

Always include headers in your data selection - they become labels in your chart!

Data Analysis Features

Excel offers powerful tools to analyze and summarize your data.

Essential Data Analysis Tools

  • Sorting: Arrange data A-Z, Z-A, or by number (smallest to largest)
  • Filtering: Show only specific data that meets criteria
  • Conditional Formatting: Automatically color cells based on values
  • Freeze Panes: Keep headers visible while scrolling
  • AutoSum: Quick button to add a range of cells
Remember

Use Ctrl+Shift+L to quickly add or remove filters from your data!
