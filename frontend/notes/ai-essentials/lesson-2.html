Neural Networks & Deep Learning

Learning Objectives

  • Understand the structure and components of neural networks
  • Learn how neurons process information
  • Explore the concept of deep learning and hidden layers
  • Understand backpropagation and how networks learn
  • Recognize different types of neural network architectures

What is a Neural Network?

A neural network is a computing system inspired by biological neural networks in the brain. It consists of interconnected nodes (neurons) that process information together to solve complex problems.

Key Components

  • Input Layer: Receives the initial data (features)
  • Hidden Layers: Process information through weighted connections
  • Output Layer: Produces the final prediction or classification
  • Weights & Biases: Parameters that the network learns during training
Did You Know?

The first neural network, called "Mark I Perceptron," was built in 1958 by Frank Rosenblatt!

How Neural Networks Work

Neural networks learn by adjusting weights based on the error in predictions. This process involves:

1. Forward Propagation

Data flows from the input layer through hidden layers to the output layer. Each neuron applies a weighted sum of inputs followed by an activation function.

2. Activation Functions

These functions introduce non-linearity, allowing networks to learn complex patterns:

  • ReLU (Rectified Linear Unit): Returns 0 for negative values, the value itself for positive
  • Sigmoid: Maps values to between 0 and 1 (useful for probabilities)
  • Tanh: Maps values to between -1 and 1
Pro Tip

ReLU is the most commonly used activation function in modern deep learning because it helps gradients flow better during training.

Deep Learning Explained

Deep learning is a subset of machine learning that uses neural networks with multiple hidden layers (hence "deep"). These deep networks can learn increasingly abstract representations of data.

Why Deep Networks Matter

  • Feature Learning: Automatically learn features from raw data without manual feature engineering
  • Complex Patterns: Capture intricate patterns in images, text, and audio
  • State-of-the-Art Performance: Achieve superior results in image recognition, NLP, and speech recognition
Important

More layers don't always mean better results. Deep networks require more data and computational power, and can suffer from vanishing gradients if not properly initialized.

Backpropagation: How Networks Learn

Backpropagation is the algorithm that allows neural networks to learn from errors. It works by:

  1. Calculate Error: Compare predictions to actual values
  2. Backward Pass: Propagate error backwards through the network
  3. Update Weights: Adjust weights to reduce error using gradient descent
  4. Repeat: Iterate until the model performs well

Training Loop

Input Data → Forward Pass → Calculate Loss → 
Backward Pass → Update Weights → Repeat

The learning rate controls how much weights change with each update. Too high = unstable training; too low = slow convergence.

Types of Neural Networks

Different architectures suit different problems:

  • Feedforward Neural Networks: Data flows in one direction; used for classification and regression
  • Convolutional Neural Networks (CNN): Specialized for image processing with convolutional layers
  • Recurrent Neural Networks (RNN): Good for sequential data like text and time series
  • Transformer Networks: Power modern language models like ChatGPT
Real-World Example

CNN architectures like ResNet achieve 99%+ accuracy on image classification tasks, surpassing human performance!
