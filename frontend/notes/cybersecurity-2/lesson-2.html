Malware & Cyber Threats

Learning Objectives

  • Understand different types of malware
  • Recognize how viruses and ransomware work
  • Identify signs of malware infection
  • Learn prevention and removal strategies

What is Malware?

Malware (malicious software) is any software designed to cause damage, steal data, or gain unauthorized access to computer systems. It's one of the most common cyber threats.

How Malware Spreads

  • Email Attachments: Infected files from suspicious emails
  • Malicious Downloads: Software from untrusted sources
  • Infected Websites: Compromised sites that exploit vulnerabilities
  • Removable Media: USB drives and external hard drives
  • Social Engineering: Tricking users into installing malware
Did You Know?

Over 350,000 new malware samples are discovered every day!

Types of Malware

Viruses

Programs that replicate themselves by attaching to other programs. They require user action to spread and can cause various damages.

Worms

Self-replicating programs that spread across networks without user intervention. They can consume bandwidth and spread rapidly.

Trojans

Disguised as legitimate software, Trojans create backdoors, steal data, or allow remote control by attackers.

Spyware

Secretly monitors user activity, collecting passwords, browsing history, and personal information.

Adware

Displays unwanted advertisements and may track browsing habits to deliver targeted ads.

Important

Ransomware

Ransomware encrypts a victim's files and demands payment for the decryption key. It's one of the most profitable and damaging malware types.

How Ransomware Works

  1. Infection through email, website, or vulnerability
  2. Silent encryption of files on the system
  3. Display of ransom note with payment instructions
  4. Demand for payment (often in cryptocurrency)
  5. Threat of data deletion if not paid

Types of Ransomware

  • Encrypting Ransomware: Encrypts files and demands payment
  • Locker Ransomware: Locks users out of their devices
  • Double Extortion: Threatens to leak stolen data
Pro Tip

The best protection against ransomware is regular, offline backups. If your files are backed up, you don't need to pay the ransom!

Signs of Malware Infection

  • Computer runs slower than usual
  • Unexpected pop-ups and browser redirects
  • New toolbars or programs you didn't install
  • System crashes or frequent error messages
  • High network usage without explanation
  • Disabled antivirus software
  • Files are missing or encrypted

Prevention and Removal

Prevention Best Practices

  • Keep all software updated with latest patches
  • Use reputable antivirus and anti-malware software
  • Be cautious with email attachments and links
  • Only download software from trusted sources
  • Use strong passwords and enable 2FA
  • Backup important data regularly

If Infected

  1. Disconnect from the network immediately
  2. Run a full antivirus scan
  3. Identify and remove the malware
  4. Restore files from backup if needed
  5. Change all passwords
  6. Report the incident if necessary
Never Pay Ransom

Paying doesn't guarantee you'll get your files back. Always report to authorities and restore from backups.
