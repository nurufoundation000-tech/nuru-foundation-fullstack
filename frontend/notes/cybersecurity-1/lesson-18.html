🛠️ 2. Lab 1 — Using VirusTotal

What is VirusTotal?

It is a free online service that analyzes suspicious files and URLs. It uses over 70 different antivirus engines (like McAfee, Kaspersky, Symantec) to check if a file is dangerous.

How it works: It creates a "Hash" (a unique digital fingerprint) of the file and compares it to a global database of known malware.

🧪 Lab 1: Step-by-Step Analysis

Follow these steps to analyze a file safely:

  1. 1. Open your browser and go to https://www.virustotal.com
  2. 2. Choose the "File" tab for documents or the "URL" tab for links.
  3. 3. Upload a file (use the sample provided by the instructor) OR paste a suspicious link.
  4. 4. Wait for the engines to process the scan.

*Note: Never upload sensitive personal documents to public scanners!

📊 3. Understanding the Results

Once the scan is finished, VirusTotal provides a detailed report:

  • Detection Ratio: (e.g., 5/70). This means 5 out of 70 engines flagged the file.
  • Behavior Tab: Shows what the file *tries* to do (e.g., "Attempts to delete system files").
  • Details Tab: Shows the file type, size, and the "MD5/SHA" fingerprints.

🚨 Interpretation: Making the Call

As an analyst, you must interpret the data:

  • 0 Detections: Likely safe (or very new "Zero-Day" malware).
  • 1-3 Detections: Suspicious. Could be a False Positive (a safe file mistakenly flagged).
  • 4+ Detections: High probability of being malicious. Do not open!
Technician Tip: Check the names given by the engines. If you see "Trojan" or "Ransom," stay away!
