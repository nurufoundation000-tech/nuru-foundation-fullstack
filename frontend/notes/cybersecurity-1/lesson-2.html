🛡️ 2. Practical 1 — Checking Real Data Breaches

What is a data breach?

A data breach is when hackers break into a system and steal information such as: Emails, Passwords, Phone numbers, and Addresses.

✅ Today’s Task: You will check if any email has ever appeared in a hack using HaveIBeenPwned (H.I.B.P).

STEPS:

  1. Open your browser and visit: haveibeenpwned.com
  2. In the search box, type an email (use an example or an old one if uncomfortable with private email).
  3. Click “pwned?”

Analyzing the Results

🚨 If it shows RED → “Oh no — pwned!”

This means:

  • Your email was found in at least one security breach.
  • Your password might be exposed.
  • You MUST change that password immediately wherever it was used!

✅ If it shows GREEN → “Good news — no breach found!”

But this does not mean you are perfectly safe — it only means your data was "not found in known hacks." Continuous vigilance is required.

🔑 4. Practical 3 — Password Strength

A weak password is easy to guess. A strong password is long, complex, and unpredictable.

Examples of WEAK Passwords (Easy to crack in <1 second)

123456 | password | kenya2024 | Alex Chen3 | iloveyou

Examples of STRONG Passwords (Requires years to crack)

Use this strong structure: Word + Symbol + Word + Number

River!Stone$992
Sky@Lion#2045
i-love-strong-coffee-everyday (Passphrase)

🧠 6. Practical 4 — Building a Security Mindset Checklist

A strong cybersecurity student must always incorporate defensive thinking into their daily digital life. This is the Security Mindset.

SECURITY MINDSET CHECKLIST (Draft)

  • ✔ I never reuse passwords
  • ✔ I update passwords every 3–6 months
  • ✔ I do not click unknown links
  • ✔ I avoid free Wi-Fi without VPN
  • ✔ I check email senders carefully
  • ✔ I avoid downloading unknown files
  • ✔ I use 2-Step Verification where possible
  • ✔ I check if my data has ever been leaked
  • ✔ I backup important files monthly

Assignment: Students will create their own 10–15 item checklist based on these principles.
