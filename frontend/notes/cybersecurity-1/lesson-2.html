Password Security & Authentication

Learning Objectives

  • Create strong, secure passwords
  • Understand multi-factor authentication (MFA/2FA)
  • Learn to use password managers effectively
  • Recognize and avoid common password mistakes

Creating Strong Passwords

A strong password is your first defense against unauthorized access. Weak passwords are responsible for 81% of data breaches.

Characteristics of Strong Passwords

  • Length: At least 12 characters long
  • Complexity: Mix of uppercase, lowercase, numbers, and symbols
  • Uniqueness: Different password for each account
  • No Personal Info: Avoid birthdays, names, or common words
Avoid These

Never use: "password", "123456", your name, birthday, or common words. Hackers can guess these in seconds!

Password Generators

Creating complex passwords manually is difficult. Password generators create random, secure passwords instantly.

How to Use Password Generators

  1. Use built-in generators in password managers
  2. Select appropriate length (12-16 characters recommended)
  3. Include all character types
  4. Avoid generated passwords that are difficult to type
Pro Tip

A good password might look like: Gk#9mP$2xLq!nR8. Random and virtually uncrackable!

Multi-Factor Authentication (MFA/2FA)

Two-Factor Authentication adds an extra layer of security by requiring something you know (password) plus something you have (phone) or something you are (fingerprint).

Types of Authentication Factors

  • Something You Know: Password, PIN, security questions
  • Something You Have: Phone, security key, smart card
  • Something You Are: Fingerprint, face recognition, voice

Common 2FA Methods

  • SMS Codes: Text messages sent to your phone
  • Authenticator Apps: Generate time-based codes (more secure than SMS)
  • Hardware Keys: Physical devices like YubiKey
  • Biometric: Fingerprint or face recognition
Did You Know?

Enabling 2FA can prevent 99.9% of account breaches, even if your password is compromised!

Password Managers

Password managers securely store all your passwords in an encrypted vault, so you only need to remember one master password.

Benefits of Password Managers

  • Generate and store unique passwords for every account
  • Automatically fill login forms
  • Sync across all your devices
  • Alert you to compromised passwords
  • Encrypt your data with strong encryption

Popular Password Managers

  • Bitwarden: Open-source, free option
  • 1Password: User-friendly, excellent security
  • LastPass: Popular with many features
  • Built-in Options: Browser password managers (less secure)
Important

Your master password should be your strongest - it's the key to all your other passwords!

Password Security Checklist

  • Change passwords immediately if you suspect a breach
  • Never share passwords via email or text
  • Use different passwords for work and personal accounts
  • Enable 2FA on all important accounts
  • Review account security settings regularly
  • Set up account recovery options in advance
