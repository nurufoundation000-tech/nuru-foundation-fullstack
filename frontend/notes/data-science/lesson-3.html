Data Visualization

Learning Objectives

  • Create visualizations using Matplotlib library
  • Build enhanced charts with Seaborn
  • Choose appropriate chart types for different data
  • Apply best practices for effective data presentation

Introduction to Data Visualization

Data visualization is the graphical representation of information and data. By using visual elements like charts, graphs, and maps, data visualization tools provide an accessible way to see and understand trends, outliers, and patterns in data.

Why Visualization Matters: Humans process visual data 60,000 times faster than text. Effective visualizations make complex data accessible and actionable.

Choosing the Right Chart

  • Line charts: Show trends over time
  • Bar charts: Compare categorical data
  • Pie charts: Show proportions of a whole
  • Scatter plots: Show relationships between variables
  • Histograms: Show distribution of numerical data
  • Heatmaps: Show density or intensity

Matplotlib Fundamentals

Matplotlib is the foundational plotting library for Python. It provides complete control over every aspect of your visualizations.

import matplotlib.pyplot as plt import numpy as np # Create simple line plot x = np.linspace(0, 10, 100) y = np.sin(x) plt.figure(figsize=(10, 6)) plt.plot(x, y, color='#3498db', linewidth=2) plt.title('Sine Wave', fontsize=16) plt.xlabel('X Axis') plt.ylabel('Y Axis') plt.grid(True) plt.show()

Creating Multiple Charts

# Create subplots fig, axes = plt.subplots(2, 2, figsize=(12, 8)) axes[0, 0].plot([1, 2, 3], [1, 4, 9]) axes[0, 1].bar(['A', 'B', 'C'], [3, 7, 5]) axes[1, 0].scatter([1, 2, 3], [3, 1, 4]) axes[1, 1].pie([30, 50, 20]) plt.tight_layout() plt.show()

Seaborn for Enhanced Visualizations

Seaborn is built on top of Matplotlib and provides a higher-level interface for creating attractive statistical graphics.

import seaborn as sns import pandas as pd # Load sample dataset tips = pd.read_csv("tips.csv") # Create a beautiful scatter plot plt.figure(figsize=(10, 6)) sns.scatterplot(data=tips, x="total_bill", y="tip", hue="day", style="smoker", s=100) plt.title('Tips by Day and Smoking Status') plt.show()

Seaborn Plot Types

  • sns.barplot(): Bar charts with error bars
  • sns.countplot(): Count frequency of categories
  • sns.boxplot(): Box-and-whisker plots
  • sns.violinplot(): Distribution visualization
  • sns.heatmap(): Correlation matrices
  • sns.pairplot(): Pairwise relationships
# Correlation heatmap corr = tips.corr() plt.figure(figsize=(8, 6)) sns.heatmap(corr, annot=True, cmap="coolwarm", center=0) plt.title('Correlation Matrix') plt.show()

Best Practices for Visualization

Design Principles:
  • Start with a clear question or message
  • Use appropriate chart types for your data
  • Keep designs simple and uncluttered
  • Use colors purposefully and consistently
  • Label axes and add titles
  • Consider your audience

Common Mistakes to Avoid

  • 3D charts that distort proportions
  • Truncated y-axes that mislead
  • Too many colors and categories
  • Pie charts with too many slices
  • Missing legends and labels

Summary

  • Data visualization transforms complex data into understandable insights
  • Matplotlib provides low-level control for any visualization
  • Seaborn offers attractive, statistical visualizations with less code
  • Choose chart types based on your data and message
  • Follow best practices for effective, truthful visualizations
