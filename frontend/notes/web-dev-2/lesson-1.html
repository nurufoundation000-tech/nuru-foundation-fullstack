JavaScript Basics

Learning Objectives

  • Understand JavaScript fundamentals and syntax
  • Learn how to declare and use variables
  • Write and call functions
  • Manipulate the DOM to update web pages

What is JavaScript?

JavaScript is a programming language that adds interactivity to websites. While HTML structures pages and CSS styles them, JavaScript makes them dynamic and interactive.

Why Learn JavaScript?

  • Universal: Runs in all modern browsers
  • Versatile: Used for front-end, back-end, mobile apps
  • In Demand: Essential skill for web developers
  • Fun: Immediate visual results
Did You Know?

JavaScript was created in just 10 days in 1995. It's now one of the most popular programming languages in the world!

Variables

Variables store data that your program can use. JavaScript has three ways to declare variables:

let (Recommended)

let name = "Alex"; let age = 25; let isStudent = true;

const (For values that won't change)

const schoolName = "NURU Academy"; const maxScore = 100;
Important

Always use const by default unless you need to reassign the variable. Use let when you need to change the value later.

Data Types

  • String: Text in quotes ("Hello")
  • Number: Numeric values (42, 3.14)
  • Boolean: true or false
  • Array: Lists of values [1, 2, 3]
  • Object: Key-value pairs {name: "Alex"}

Working with Strings

let greeting = "Hello, "; let name = "World"; let message = greeting + name; // "Hello, World" // Template literals (use backticks) let fullMessage = `${greeting}${name}!`;

Functions

Functions are reusable blocks of code that perform a specific task.

Function Declaration

function greet(name) { return "Hello, " + name + "!"; } // Call the function let result = greet("Alex");

Arrow Functions (Modern Syntax)

const add = (a, b) => a + b; const square = (x) => x * x;

DOM Manipulation

The DOM (Document Object Model) is how JavaScript interacts with HTML elements.

Selecting Elements

// By ID const heading = document.getElementById("myHeading"); // By class const cards = document.getElementsByClassName("card"); // Using querySelector const firstButton = document.querySelector("button");

Changing Content

// Change text heading.textContent = "New Title"; // Change HTML element.innerHTML = "<strong>Bold</strong>"; // Change styles element.style.color = "red";
Pro Tip

Always wrap your JavaScript in an event listener or place it at the bottom of the body to ensure the HTML exists before your code runs.
