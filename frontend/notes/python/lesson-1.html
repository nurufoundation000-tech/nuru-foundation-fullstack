Introduction to Python Programming

Learning Objectives

  • Understand what Python is and why it's popular
  • Set up your Python development environment
  • Write and run your first Python program
  • Learn basic Python syntax and print statements

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum in 1991, Python has become one of the most popular programming languages in the world.

Why Learn Python?

  • Easy to Learn: Python has a simple syntax that resembles English, making it perfect for beginners
  • Versatile: Used in web development, data science, AI, automation, and more
  • High Demand: Python developers are among the most sought-after in the job market
  • Large Community: Thousands of libraries and frameworks available
Did You Know?

Python is used by companies like Google, Netflix, Instagram, and Spotify!

Setting Up Your Environment

Before writing Python code, you need to install Python on your computer.

Installing Python

  1. Go to python.org
  2. Click the "Downloads" button
  3. Download the latest version for your operating system
  4. Run the installer and check "Add Python to PATH"
  5. Click "Install Now"

Choosing a Code Editor

You can write Python code in various editors:

  • IDLE: Comes with Python installation
  • VS Code: Free, powerful, with Python extension
  • PyCharm: Professional IDE for Python
  • Jupyter Notebook: Great for data science
Pro Tip

For beginners, we recommend starting with VS Code + Python extension. It's free and provides helpful features like code completion and error highlighting.

Your First Python Program

Let's write the classic "Hello, World!" program - a tradition for learning any programming language.

The print() Function

In Python, we use the print() function to display text on the screen.

# This is a comment - Python ignores this line print("Hello, World!")

When you run this code, you should see:

Hello, World!

Running Your Code

  1. Open your code editor (VS Code, IDLE, etc.)
  2. Type the code above
  3. Save the file as hello.py
  4. Run the program (usually F5 or a Run button)
Important

Python files should always end with .py. This tells your computer that the file contains Python code.

Understanding Python Syntax

Python has unique rules for writing code, known as syntax. Let's learn the basics.

Comments

Comments are notes for humans reading the code. Python ignores comments.

# This is a single-line comment """ This is a multi-line comment. You can write longer explanations here. """

Strings

Text in Python is called a "string" and must be wrapped in quotes:

print("Double quotes work") print('Single quotes also work') print("You can mix them if you need to include a quote: 'like this'")

Numbers

Python can do math directly:

print(2 + 2) # Addition: 4 print(10 - 3) # Subtraction: 7 print(4 * 3) # Multiplication: 12 print(15 / 3) # Division: 5.0

Now it's your turn! Try creating a program that:

  1. Prints a greeting with your name
  2. Prints your age
  3. Calculates 25 + 17
Expected Output
